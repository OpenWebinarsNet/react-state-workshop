# Ejecutar el proyecto

Simplemente clonarse este repositorio.

##### Instalar dependencias

```bash
npm i
```

##### Ejecutar proyecto

```bash
npm start
```

Por último solo hay que ir a http://localhost:3000 para ver la aplicación
